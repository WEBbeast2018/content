### Angular Unit Testing
[Jasmine](https://jasmine.github.io/) - Behavior-Driven JavaScript
```
describe("A suite is just a function", function() {
  let a;

  it("and so is a spec", function() {
    a = true;

    expect(a).toBe(true);
  });
});
```
